圓方程 - 求圓心和半徑
Finding Center and Radius from Circle Equations
===================================

Level number：4

功能：
這是一個用於生成圓方程題目的生成器，要求學生從給定的圓方程中求出圓心坐標和半徑。題目涵蓋了圓的標準形式方程、一般形式方程以及需要代數變換的複雜形式方程。生成器提供了不同難度級別的題目，從基礎到高級，逐步增加挑戰性。

難度級別：
1. 基礎：圓的標準形式方程，圓心為非負整數坐標，半徑為整數。
   - 例如：(x-3)² + (y-4)² = 25
2. 進階：圓的標準形式方程，圓心可能有負坐標，半徑為整數。
   - 例如：(x+2)² + (y-5)² = 16
3. 高級：圓的一般形式方程，需要學生從一般形式中找出圓心和半徑。
   - 例如：x² + y² - 4x + 6y + 9 = 0
4. 挑戰：複雜形式方程，可能包含係數，需要學生進行代數變換。
   - 例如：3x² + 3y² + 12x - 18y + 21 = 0

題目特點：
- 每個難度級別提供適合的方程式類型，循序漸進。
- 標準形式方程為：(x-h)² + (y-k)² = r²，其中(h,k)是圓心，r是半徑。
- 一般形式方程為：x² + y² + 2gx + 2fy + c = 0，其中圓心為(-g,-f)，半徑為√(g²+f²-c)。
- 複雜形式需要學生先進行變換，例如將ax² + ay² + 2agx + 2afy + ac = 0轉換為標準或一般形式。

輸出格式：
- 問題文本：清晰地呈現圓方程，並提示學生如有必要使用根式表示答案。
- 正確答案：以"圓心 = (h, k), 半徑 = r"的格式呈現，其中h、k和r為相應的數值。
- 錯誤答案：包含常見的計算錯誤，例如圓心坐標符號錯誤或半徑與半徑平方混淆。
- 解釋：詳細的解題步驟，包括方程識別、係數提取、完全平方和方程轉換等過程。

適用技能：
- 識別圓的不同形式方程（標準形式、一般形式和複雜形式）
- 從圓方程中提取圓心坐標和半徑
- 應用完全平方法將一般形式轉換為標準形式
- 處理帶係數的方程，進行必要的代數變換

教學目標：
- 理解圓的標準形式方程與圓心、半徑的關係
- 熟練掌握圓的一般形式方程與圓心、半徑的關係
- 能夠處理各種類型的圓方程，提取關鍵參數
- 發展代數變換和方程操作的技能
- 提高對幾何圖形代數表示的理解
